> 프로젝트 소개
- 사용자가 특정 지하철역 까지의 소요시간을 입력하면, 대중교통으로 해당 소요시간 내에 도달 가능한 아파트 매물 정보 제공

## 핵심 기능  Key Feature
- 사용자가 원하는 소요시간 내에 대중교통으로 도달 가능한 아파트 매물 정보를 제공합니다.
- 매물의 단지명, 시세를 제공합니다.
## 사용 How To Use
- 강남구 아파트지만 강남역까지 30분 걸리는 아파트(25평, 14억) vs 경기도 아파트지만 강남역까지 30분 걸리는 아파트(25평, 9억)
